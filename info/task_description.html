The spaceship reactors contains several (no more than ten) energy nuclear rods. These rods should be sorted by size for stable reactor working. But as usual the stupid reactor serving machine had placed all nuclear rods without any order and started the engine. Now Stephan have to reorder all rods. Because the reactor is working already he can only quick swap neighbour rods or BOOM!

You are given sizes and the initial order of rods as an array of numbers. Indexes are position, values are sizes. You should order this array from the lowest to the biggest sizes.

For each action Stephan can swap only two neighbour elements. Each action should be represented as a string with two digits - indexes of the swapping elements (ex, "01" - swap 0th and 1st rods). The result should be represented as a string that contains the sequence of actions separated by commas. If the array is not required to be sorted, then return an empty string.

And you can swap only N*(N-1)/2 times, where N - is a quantity of rods.

Input: An array as a tuple of integers.

Output: The sequence of actions as a string.

Example: 

robosort((6, 4, 2)) == "01,12,01"
robosort((1, 2, 3, 4, 5)) == ""
robosort((1, 2, 3, 5, 3)) == "54"

How it is used: This mission will show you how to work the simplest sorting algorithms. And this can be useful, if you need to build a production line for your factory.

Precondition:
1 ≤ len(array) ≤ 10
all(1 ≤ n < 10 for n in array)
