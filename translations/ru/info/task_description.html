Реактор космического корабля содержит несколько (не более десяти) ядерных топливных стержней. Эти стержни должны быть отсортированы по размеру, чтобы обеспечить стабильность работы реактора. Но, как обычно это бывает - двигатель корабля уже запущен, а механизм загрузки топливных стержней перед этим дал сбой и стержни были установлены в случайном порядке. Теперь Стефан должен перезагрузить все топливные стержни сначала. Но поскольку реактор уже работает, то ему необходимо максимально быстро поменять соседние стержни, иначе реактор взорвётся!

Вам даны размеры и первоначальный порядок стержней, как массив чисел. Индекс - позиция стержня, а значение - это его размер. Вы должны упорядочить этот массив по размеру - от самых маленьких стержней, до самых больших.

За один раз Стефан может поменять местами только два соседних элемента. Каждое действие должно быть записано в виде строки с двумя цифрами - индексы меняемых местами элементов (например, "01" - означает, что 0-й и 1-й стержень поменяли местами). Результат должен быть представлен в виде строки, которая содержит последовательность действий, разделенных запятыми. Если массив не требует сортировки, тогда необходимо вернуть пустую строку.

Производить обмен вы можете только N*(N-1)/2 количество раз, где N - это количество стержней.

На входе: Массив, как кортеж целых чисел.

На выходе: Последовательность действий в виде строки.

Например: 

swapsort((6, 4, 2)) == "01,12,01"
swapsort((1, 2, 3, 4, 5)) == ""
swapsort((1, 2, 3, 5, 3)) == "43"

Как это используется: Эта миссия покажет вам, как работать с простейшими алгоритмами сортировки. И это может быть полезно, если вам вдруг необходимо построить производственную линию для вашего завода.

Предусловия:
1 ≤ len(array) ≤ 10
all(1 ≤ n < 10 for n in array)
